Python — це потужна та проста у використанні мова програмування, яка підходить для початківців. Ось кілька основних моментів для новачків:

Розділ 2

У Python можна легко працювати зі списками. Ось приклад, як додати елементи до списку та вивести їх на екран:

# Приклад роботи зі списками
my_list = [1, 2, 3]
my_list.append(4)  # Додаємо 4 до списку
print(my_list)     # Виведе: [1, 2, 3, 4]

Розділ 3

Python також підтримує створення класів. Ось простий приклад класу та об'єкта:

# Приклад об'єктно-орієнтованого програмування
class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        return f"{self.name} каже гав!"

my_dog = Dog("Рекс")
print(my_dog.bark())  # Виведе: Рекс каже гав!
